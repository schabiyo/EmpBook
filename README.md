# EmpBook

Experience a new way of managing employees with ReactJS, Spring Boot and MongoDB.

![EmpBook](/img/empbook.png "EmpBook")

### Step 1 - Pre-requisite

You need the following:

- **Maven**
- **MongoDB** 


### Step 2 - Run the App

` $ git clone https://github.com/schabiyo/EmpBook.git `

` $ cd EmpBook `

` $ mvn spring-boot:run `
